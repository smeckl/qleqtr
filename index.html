Welcome to Qleqtr.

Qleqtr is a host-based collection agent for Windows. It is a Windows Service developed in C# allowing clients to securely connect to a Windows system using a JSON-based API to collect system artifacts and indicators of compromise. The tool is in an early stage of development. Updates will be posted to this as new features are added or changes are made.
